# LoveCoder
爱媛猿app 开始构建中
### 主要功能:
* 本地天气（温度、空气质量）动态展示，其他信息以弹幕形式展示。
* 萌妹图配视频文字动态展示。
* 萌妹图浏览查看保存等图片操作。
* 视频播放。
* 文字信息浏览。
